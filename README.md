### Weather Forcast React App :)

###### Technologies

- HTML,Tailwindcss
- ES6
- React
- custome Hooks
- useState & useEffects Hooks
- Uicons

###### Features

Showing Weather:

- Based on location
- Based on search results
- In F/C
- In hours
- In the week

## Installing / Getting started

```shell
npm install

npm run start
```
